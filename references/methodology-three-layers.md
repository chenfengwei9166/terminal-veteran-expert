## 第一层：分析层 — 3条数据分析法

### 法则1：市场数据解读法

**核心口诀**：看份额不看销量、看价位段不看总份额、看3月MA不看单月

**操作步骤**：
1. 判断季节位置：当前月份处于铁律的哪个阶段？（峰值/低谷/常态）
2. 对比铁律基准：数据偏离铁律多少？偏离才是异常信号
3. 3月移动平均：用MA3平滑春节/苹果周期扭曲
4. 交叉验证：份额异动 + 价位段变化 + 渠道结构变化是否一致

**典型误判警示**：
- 看到单月同比负增长就喊"市场衰退"（可能是春节错位）
- 看到总份额变化就调整策略（可能是价位段内正常波动）
- 苹果9月发布后就判断安卓不行了（正常周期回落）

### 法则2：品牌×价位段山头分析法

**核心口诀**：先定位"山头"再看份额，"窄而深"胜过"宽而薄"

**操作步骤**：
1. 画山头地图：各价位段的统治品牌是谁？统治份额多少？
2. 识别混战区：哪个价位段没有绝对统治者？（当前：2-3K）
3. 盯信号品牌：混战区份额波动最大的品牌=选题信号/策略信号
4. 评估进攻可行性：目标价位段的山头是否稳固？有无被颠覆迹象？

**山头地图（2026基准）**：

| 价位段 | 统治品牌 | 统治份额 | 稳固度 | 信号强度 |
|--------|---------|---------|--------|---------|
| 1.5-2K | 小米 | 32% | 高 | 低 |
| 2-3K | vivo/荣耀 | ~22% | 低（混战） | **最高** |
| 3-4K | OPPO | 42% | 高 | 低 |
| 4-5K | 华为 | 25% | 中 | 中 |
| 5K+ | 苹果 | 57% | 极高 | 低 |

### 法则3：线上线下渠道差异分析法

**核心口诀**：线下是1.8倍主体，线上卖贵手机线下卖便宜手机

**操作步骤**：
1. 确认渠道占比：线上%/线下%（当前：35.1% / 64.9%）
2. 对比渗透率差：线上5G渗透率 vs 线下5G渗透率（当前：89% vs 76%）
3. 识别品牌渠道基因：该品牌线上强还是线下强？
4. 匹配联通渠道优势：联通线下渠道适配哪些品牌？

**品牌渠道基因表**：

| 品牌 | 线上基因 | 线下基因 | 联通适配度 |
|------|---------|---------|-----------|
| 小米 | 强 | 弱 | 线上合作 |
| 苹果 | 强 | 中 | 全渠道 |
| 华为 | 中 | 强 | **线下重点** |
| OPPO | 弱 | 强 | **线下重点** |
| vivo | 弱 | 强 | **线下重点** |

## 第二层：操盘层 — 10套运营方法论

### M1 分类分级运营模型

**核心逻辑**：根据终端与运营商业务的耦合程度，差异化配置资源

**5G时代三级目录**：
- 爆款目录：销量爆款、明星产品 → 最大资源
- 重点目录：重点推广产品 → 较大资源
- 常态目录：常规产品 → 基础资源

### M2 品牌五维分类法

**五维分类**：

| 类型 | 战略重要性 | 合作深度 | 资源投入 | 2017年代表 |
|------|-----------|---------|---------|-----------|
| 战略品牌 | 最高 | 全方位 | 最大 | 苹果、三星、华为 |
| 重点品牌 | 高 | 重点产品 | 大 | OPPO、vivo、小米 |
| 潜力品牌 | 中 | 成长性 | 中等 | 魅族、一加 |
| 一般品牌 | 低 | 常规 | 小 | 其他国产 |
| 观察品牌 | 待观察 | 观察 | 观察 | 新兴品牌 |

### M3 "四个面向"决策框架

**操作模板**：

```
【决策主题】：________________

一、面向变化
1. 技术变化：________________
2. 政策变化：________________
3. 竞争变化：________________
4. 用户变化：________________
→ 输出：变化清单 + 影响评估

二、面向问题
1. 核心问题：________________
2. 问题根因：________________
3. 问题优先级：________________
→ 输出：问题清单 + 优先级排序

三、面向大市场
1. 机会识别：________________
2. 机会大小：________________
3. 可行性评估：________________
→ 输出：机会清单 + 可行性评估

四、面向未来
1. 布局方向：________________
2. 关键举措：________________
3. 时间表：________________
→ 输出：行动计划 + 时间表

【交叉验证】四个面向结论是否一致？如有矛盾，优先"面向未来"
```

**陈总应用案例**：2016年预判eSIM
- 面向变化：全网通趋势→制式差异化消失
- 面向问题：补贴受限→需要新差异化手段
- 面向大市场：物联网万亿级市场
- 面向未来：AI时代连接形态变革
→ 结论：押注eSIM

### M4 "三体验为主"运营框架

| 体验维度 | 对象 | 核心诉求 | 体验差的后果 |
|----------|------|---------|-------------|
| **上游体验** | 厂商/供应商 | 合作简单化 | 厂商不愿合作→产品匮乏 |
| **下游体验** | 渠道/零售商 | 赚钱好卖 | 渠道不愿卖→销量下滑 |
| **用户体验** | 最终用户 | 高品质好用 | 用户不满意→品牌受损 |

### M5 "4P皆互联网化"

| 4P维度 | 传统模式 | 互联网化模式 |
|--------|---------|-------------|
| **Product** | 深度定制、运营商专供 | 标准产品为主、轻度定制、全网通 |
| **Price** | 补贴压价、套餐捆绑 | 透明定价、金融分期、权益融合 |
| **Place** | 国代+自有、层级分销 | 线上线下融合、社交电商、直播 |
| **Promotion** | 广告投放、地推活动 | 精准营销、数据驱动、社交裂变 |

### M6 "融终端"模式

**核心公式**：融终端 = 平台 + 网 + 终端 + X

| 组件 | 说明 | 运营商优势 |
|------|------|-----------|
| **平台** | 运营商终端运营平台 | 已建立 |
| **网** | 5G网络能力 | 核心优势 |
| **终端** | 泛智能终端 | 渠道优势 |
| **X** | 业务、金融、内容等增值服务 | 整合能力 |

**五大市场实施路径**：

| 市场 | 融终端公式 | 案例 |
|------|-----------|------|
| 政企要客 | 平台+网+终端+云服务 | 5G+云+定制终端 |
| 楼宇商企 | 平台+网+终端+IoT | 宽带+CPE+传感器 |
| 个人 | 平台+网+终端+金融 | 手机+分期+权益包 |
| 家庭 | 平台+网+终端+内容 | 宽带+路由器+视频 |
| 乡村 | 平台+网+终端+以旧换新 | 下乡手机+旧机抵扣 |

### M7 终端产品棋盘

**核心公式**：终端产品棋盘 = 终端(棋子) + 价值(棋盘) + 分级分类匹配(棋谱)

**棋子分级**：
- 爆款目录：严格控制数量，匹配最高资源
- 重点目录：适度控制，匹配核心资源
- 常态目录：广泛覆盖，匹配基础资源

**棋盘：23项价值清单**（营销资源/渠道资源/金融支持/套餐权益/库存保障等）

### M8 号卡转化率提升方法论

**2019年5G登网率数据**：
- 自有渠道：60%
- 社会渠道：22%
- 全渠道：25%

### M9 "三个一定要"

| 一定要 | 核心要求 | 目的 |
|--------|---------|------|
| **一定要登网** | 销售时必须完成5G网络激活 | 避免"5G手机用4G网" |
| **一定要转套餐** | 引导用户迁转5G套餐 | 实现ARPU提升 |
| **一定要体验应用** | 让用户体验5G特色应用 | 建立5G价值感知 |

### M10 "机网协同"原则

**5G时代教训**：社会渠道5G登网率仅22%
- 原因：只强调"5G手机"，不强调"5G网络覆盖"
- 后果：用户买了5G手机但没用5G网络

**AI时代延伸**：不仅是"机网协同"，而是"端-边-云协同"
- AI终端需要端侧算力+边缘算力+云端算力的协同
- 销售AI终端时必须同步强调网络能力（失网即失智）

## 三层联用指南

### 场景A：视频号口播选题（5分钟出稿）

1. **分析层找数据**：查1号数据手册→选一个有数据的选题方向
2. **操盘层找方法**：查2号方法论→选一个能解释数据的方法论
3. **战略层找高度**：查3号方法论→升维到战略判断，结尾金句收尾

### 场景B：行业署名文章（30分钟出框架）

1. **战略层定调子**：选一个S级方法论作为文章核心论点
2. **分析层给数据**：用3条数据分析法解读市场数据支撑论点
3. **操盘层给案例**：用2号决策案例或方法论作为实践佐证

### 场景C：论坛演讲（1小时出完整PPT框架）

1. **战略层开场**：用一个S级方法论定全场基调
2. **分析层展开**：数据+规律支撑
3. **操盘层落地**：方法论+案例
4. **战略层收尾**：金句升华

### 场景D：8号专家设计训练

**注入顺序**：
1. 先注入分析层（3条数据分析法）→ 让专家会看数据
2. 再注入操盘层（M3/M6/M10核心）→ 让专家会干业务
3. 最后注入战略层（S1-S5核心）→ 让专家会判方向
4. 统一金句库 → 让专家会说话
